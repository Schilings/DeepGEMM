## PRE BWD 改用融合 A2A+GEMM 算子（bf16_a2a_transpose_gemm_nt, M0）

## 重构为 autograd.Function + 真实 FSDP2 (进行中)
- 用户严厉指出：融合算子要用于训练就必须封装成 torch.autograd.Function，让 forward/backward 走 autograd graph，FSDP2(fully_shard) 才能自动触发梯度 reduce-scatter。手写 backward 是错误的。
- 调研结论：FSDP2 可用 via `from torch.distributed.fsdp import fully_shard`（composable API，DTensor-based）。不是 `torch.distributed.fsdp2` 模块。PyTorch 2.9。
- FSDP2 原理：fully_shard 把参数转成 DTensor(sharded dim-0)，forward pre-hook 自动 all-gather，backward hook 自动 reduce-scatter 梯度。ignored_params 可跳过某些参数(fused_var Wo 行切分)。
- 关键坑：`strat.forward(x)` 直接调 forward 方法**绕过 __call__**，FSDP2 pre-hook 不触发！必须用 `strat(x)` 走 __call__。
- 已完成：
  1. autograd_ops.py：三个 autograd.Function（GemmA2ATransposeFunction, A2ATransposeGemmFunction, GemmRSFunction），forward 用 fused kernel，backward 用对偶 kernel + 手动算权重梯度。
  2. serial.py：NCCLAllToAll 包装成 autograd.Function（backward=A2A-inverse）。
  3. base.py：移除手写 backward()/_attn_backward/_post_backward/_pre_backward，forward 走 autograd graph。
  4. fused_standard.py / fused_variant.py：_pre_forward/_post_forward 调 autograd.Function。
  5. bench_wan21_fsdp2.py：新 bench，strat() 调用方式，autograd.backward()，FSDP2 自动梯度同步。
- 当前状态：FWD 通过（FSDP2 pre-hook 正确 all-gather 参数），BWD 失败——aten.mm mixed DTensor。原因：backward 时参数被 reshard 回 DTensor，但 autograd 计算权重梯度时需要 full 参数。FSDP2 的 backward pre-hook 应该 re-all-gather 但可能没注册或没触发。
- 下一步：调研 FSDP2 backward hook 机制，可能需要 reshard_after_forward=True（当前已设）+ 确认 backward pre-hook 注册。或用 unshard/reshard API 手动管理。

## FSDP2 + autograd.Function 重构完成
- 关键坑2：`reshard_after_forward=False` 必须设！True 会在 forward 后 reshard 参数回 DTensor，导致 backward 时 autograd 计算权重梯度遇到 mixed DTensor 错误。False 保持参数 unsharded。
- 关键坑3：NCCLAllToAll.backward 的 grad_recv 需要 `.contiguous()`（autograd 传入的 grad 可能不连续）。
- 关键坑4：autograd.Function 的 forward 和 backward 用不同的 sym buffer！GEMM+A2A forward 用 GemmA2ATransposeSymmBuffer，backward(A2A+GEMM) 用 BF16A2ATransposeGemmSymmBuffer（有 .x/.gathered）。GEMM+RS forward 用 GemmRSSymmBuffer，backward(AG+GEMM) 用 BF16AGGemmSymmBuffer（有 .x）。需要传两个 buffer。
- 2卡 verify serial+fused_std 全 PASS（fwd_rel~0.0016, bX_rel~0.0019）。
- 8卡 bench（FSDP2 autograd）：三策略全跑通。fused_var F+B 全 shape 最快（1x32K 1.05x, 1x168K 1.01x）。
- FSDP2 完全自动管理梯度同步：`y.backward(gy)` 自动触发 reduce-scatter，无需手动 all-reduce。fused_var 的 Wo 用 ignored_params 跳过。
